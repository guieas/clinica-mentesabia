# Site da Clínica Mente Sábia

Site institucional para clínica de saúde mental.

## Como usar

1. Edite o arquivo `index.html` e atualize as informações:
   - Nome da clínica
   - Endereço, telefone e e-mail
   - Horários de atendimento
   - Número do CRP

2. Abra o arquivo `index.html` no navegador para visualizar

## Como hospedar gratuitamente

### Opção 1: GitHub Pages
1. Crie uma conta no GitHub
2. Crie um repositório novo
3. Faça upload dos arquivos (index.html, styles.css, script.js)
4. Vá em Settings > Pages
5. Selecione a branch main e salve
6. Seu site estará disponível em: `https://seuusuario.github.io/nome-do-repositorio`

### Opção 2: Netlify
1. Acesse netlify.com
2. Arraste a pasta do projeto para o site
3. Pronto! Seu site estará no ar

### Opção 3: Vercel
1. Acesse vercel.com
2. Importe o projeto
3. Deploy automático

## Personalização

- Cores: edite as variáveis CSS em `styles.css` (`:root`)
- Conteúdo: edite o `index.html`
- Funcionalidades: adicione em `script.js`
